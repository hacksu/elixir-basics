    ListServer

    ListServer stores a list as the state of a process, and provides functionality to interact with that list in basic ways.

    Summary

    Functions

    add(pid, item)

    Add an item to the list

    get(pid)

    Retreives the current state of the list server

    remove(pid, item)

    Remove an item from the list
